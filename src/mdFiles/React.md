# React

## 为什么学

``` shell
原生JS操作DOM繁琐，效率低，并且浏览器会进行大量的重绘重排。

原生JS没有组件化编码方案， 代码复用率低

React 才用组件化模式，声明式编码，提高开发效率和组件复用率。在Reactnative中 使用React语法进行移动端开发。 使用虚拟DOM和Diff算法尽量减少和真实DOM之间的交互

babel: JsX -> js   es6->es5

JSX:   是React.createElement创建虚拟dom的语法糖
```
